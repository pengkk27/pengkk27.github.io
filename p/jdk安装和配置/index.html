JDK简单介绍

  • JDK Java开发工具包
  • JRE Java运行环境
  • JVM Java虚拟机

下载JDK

Orale官网 华为开源镜像站

由于在国内官网的下载速度比较慢，所以建议用华为镜像站来进行下载JDK。

安装JDK

安装JDK

选择JDK所要安装到的文件目录

安装完成

还有一种情况是压缩包的形式，只要将压缩包下载之后进行解压就可以继续下面的步骤。

配置Java环境变量

右击此电脑，点击属性，进入高级系统设置

点击高级下面的环境变量

在下面的系统变量中找到path，双击进入

点击新建，将刚才安装JDK的目录复制进去，点击确定 (这里其实是放的JAVA_HOME，当然，这样放也是没有问题的)

在系统变量中点击新建， 变量名为 classpath 变量值为 .%JAVA_HOME%\lib\tools.jar;%JAVA_HOME%\lib\dt.jar

再次新建， 变量名为 JAVA_HOME 变量值填入JDK安装的地址

打开命令行 输入

javac

然后回车，如图显示，即为配置完成

接着输入

java

回车，如图显示即为配置完成
