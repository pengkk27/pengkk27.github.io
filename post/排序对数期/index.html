排序 算法

在学习排序算法的时候，通常需要进行预先设置一个数列，来对其进行排序。但是这个人为设置的数组比较小，对整个排序算法的测试并不是很全面。所以可以编写一个对数器，来对排序算法进行验证。

对数器设计

  • 为了做到随机的产生一个数组，我们可以Java自带的Random类来产生随机数。
  • 我们需要将这个数复制一份，然后使用Java自己的数组工具来对数组进行排序，然后我们再使用我们自己写的算法来对其进行排序
  • 将两个序列排序好之后，我们再将这两个序列对应的位置进行比较。

通过这个步骤，我们就可以来验证自己写的排序算法是否正确。


import java.util.Arrays;
import java.util.Random;

public class DataChecker {

    public static int[] generateRandomArray() {
        Random r = new Random();

        int[] arr = new int[10000];

        for (int i = 0; i < arr.length; i++) {
            arr[i] = r.nextInt(10000);
        }

        return arr;
    }

    public static void check() {
        int[] arr = generateRandomArray();
        int[] arr2 = new int[arr.length];

        System.arraycopy(arr, 0, arr2, 0, arr.length);

        Arrays.sort(arr);
        YourSort.sort(arr2);

        boolean same = true;
        for (int i = 0; i < arr.length; i++) {
            if (arr[i] != arr2[i]) {
                same = false;
            }
        }

        System.out.println(same == true ? "right" : "wrong");
    }

    public static void main(String[] args) {
        check();
    }
}
