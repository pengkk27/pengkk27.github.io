Java定制自己的工具类

Aug 1, 2021
1 min read
Java 技巧

很多的Java程序员都由一套自己常用的工具类。为了编写代码时的方便，可以编写自己的工具类，让自己的开发变得简洁方便。

1、新建一个包

打开IDEA的工程后，新建一个包，然后在包下创建一个java文件。 然后在这个Java文件中写入方法。 这里使用简单的打印 hello 的方法。 为了在之后的调用方便。这里的方法设置为静态方法。

2、打包文件为jar

在设置中点击Project Structure。 然后如图按顺序进行操作。 选中你要进行包装的module。然后确定。 然后在Build中选中Build Artifacts。 选择Build或者Rebuild

3、导入

为了可以调用这个包内的方法，我们需要把包添加到Libraries中。

选中你的这个工程文件。我这里是pengkk27。 然后确定。

4、查看是否成功

这时候新建一个Test.java文件。静态的导入test.Hello.hello包。 然后运行查看是否成功。
