一个什么都不会的小菜鸟

Windows Terminal

Aug 1, 2021
1 min read
Windows 技巧

之前在Linux的课程上，对命令终端有了一定的了解。而且Linux终端可以通过oh-my-zsh来进行美化。但我那个时候使用的是fish，oh-my-fish来对终端进行美化。而且还有命令提示的功能。

但是在Windows上，命令行终端只有cmd和PowerShell两个。而且这两个的界面都不是很好看。所以我找到了Windows发布的Windows Termianl。

根据Windows官网的文档进行了安装美化。

1. 下载

下载Windows Terminal可以通过Microsoft Store来下载。 也可以通过GitHub上面的地址来下载。这里我就不列出来了。可以通过去官网找到。

2. 设置

Windows Terminal的设置可以通过json文件来设置。即settings.json文件。

在里面可以设置整个终端的所有属性。像主题之类的。都可以在里面自己配置。

2.1 安装字体

这里有个问题，官网上说的是安装Cascadia Code PL 或 Cascadia Mono PL这两个字体。不然使用其他的会出现乱码的问题。 但我安装之后使用，还是出现了一些图标没有加载。于是我在Stackoverflow上找到了问题的解。

解决方法

他的描述大概就是：在PowerShell中图标无法渲染。 这是因为新版的Cascadia Code PL字体已经不支持这些图标了。所以解决方法是使用Cascadia code nerd fonts补丁。

2.2安装Posh-Git和Oh-My-Posh

这里在PowerShell里分别执行这两个命令进行安装：

Install-Module posh-git -Scope CurrentUser
Install-Module oh-my-posh -Scope CurrentUser

2.3自定义PowerShell提示符

上面的步骤都做完之后，需要创建PowerShell配置文件，这里也使用命令：

New-Item -ItemType File -Path $PROFILE -Force

然后打开这个配置文件

notepad $PROFILE

然后就会在记事本中打开，然后再将下面的内容复制到文件中

Import-Module posh-git
Import-Module oh-my-posh
Set-Theme Paradox

这里又会出现一个问题。在打开PowerShell的时候，会提示报错。是Set-Theme这里出错 这里需要再次修改，只需要修改最后一行。这其实是对Oh-My-Posh的主题配置。最后面对单词就是主题的名称。这里可以自己选择。

Set-PoshPrompt -Theme agnosterplus

2.4最后就是将setting.json中的字体进行修改

{
    // Make changes here to the powershell.exe profile.
    "guid": "{61c54bbd-c2c6-5271-96e7-009a87ff44bf}",
    "name": "Windows PowerShell",
    "commandline": "powershell.exe",
    // "fontFace": "Cascadia Code PL",这是官网上的，因为字体已经不支持图标了，所以就要那个补丁。
    "fontFace": "CascadiaCode Nerd Font",
    "hidden": false
},

到这里，对Windows Terminal的所有美化配置都完成了。
