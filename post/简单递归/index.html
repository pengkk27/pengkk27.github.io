递归 算法

直接或间接的调用自身，这就是递归。

递归方法必须要有一个终止条件，否则会一直调用下去。而且每个递归开始都要有一个非递归定义的初始值来进行计算。

1. 例子1：

我们将常见的累加运算，我们可以使用for循环来实现，从1加到n.

public static int add(int n) {
    int sum = 0;
    for (int i = 0; i <= n; i++) {
        sum += i;
    }
    return sum;
}

使用递归则代码变为：

public static int add(int n) {
    if (n < 1)
        return -1;
    if (n == 1)
        return 1;
    return n + add(n - 1);
}
  • 假如我们要计算5的累加，则将5传入进去。if判断都不成立，就执行最后的return语句，为5 + add(4)
  • 然后参数为4，执行该方法，还是一样不满足判断，直接return 4 + add(3)
  • 然后都是一样，3 + add(2) 、 2 + add(1)
  • 到参数为1的时候，返回为1，这时候就开始往上加，通过这样来实现累加
  • 得出最后结果为15

阶乘函数是一个数学的递归公式，根据这个公式的判断条件来看。 当n为0时，给出了递归的初始值。

public static int factorial(int n) {
    if (n == 0)
        return 1;
    return n * factorial(n - 1);
}
