数组工具

数组的打印这里我自己写了一个小方法，在编写算法的时候，使用 print(arr) 可以快速的打印出数组的内容。

当然打印数组也可以使用Java自带的 Arrays.toString() 这个方法。

在自己编写调试的时候，写 print 比写 Arrays.toString() 这几个单词要快一点。所以在这里编写了这个“多此一举”的方法。

public class ArraysTools {
    public static void print(int[] arr) {
        for (int i = 0; i < arr.length - 1; i++) {
            System.out.print(arr[i] + " ");
        }
        System.out.println(arr[arr.length - 1]);
    }

    public static void swap(int[] arr, int i , int j) {
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }
            
    public static int[] findMaxMin(int[] arr) {
        int max = Integer.MIN_VALUE;
        int min = Integer.MAX_VALUE;
        for (int num : arr) {
            max = Math.max(max, num);
            min = Math.min(min, num);
        }
        return new int[]{max, min};
    }

    public static int findMax(int[] arr) {
        int max = Integer.MIN_VALUE;
        for (int num : arr) max = Math.max(max, num);
        return max;
    }

    public static int findMin(int[] arr) {
        int min = Integer.MAX_VALUE;
        for (int num : arr) min = Math.min(min, num);
        return min;
    }
}

1. 选择排序

实现思路

  • 先从整个数列中找到一个最小 ( 大 ) 的数，然后将它与整个序列的第一个（最后一个）进行交换；
  • 然后从剩下的数列中重复第一个操作，直到所有的数都进行了排序。

代码实现

public void selectSort(int[] arr) {
    for (int i = 0; i < arr.length - 1; i++) {
        int min = i;
        for (int j = i + 1; j < arr.length; j++) {
            if (arr[min] > arr[j]) {
                min = j;
            }
        }
        ArraysTools.swap(arr, i, min);
    }
}

2. 冒泡排序

实现思路

  • 将数组的第一个数与后一个数比较，如果小（大），则进行交换；
  • 然后对将第二个与后一个进行比较，如果小（大），则进行交换；
  • 类似的，将三个、第四个都与后面的进行比较，然后交换或不交换，当数组的所有数都进行了这个操作之后，最大（小）的数就会到最后一个位置。
  • 然后又从头开始，再次重复上的操作，直到最后所有的数都进行了排序

代码实现

这里的第二个for里面的判断条件减i，是因为，每次排序之后，最后的i个顺序都已经时排好的，不需要排序了

public void bubbleSort(int[] arr) {
    for (int i = 0; i < arr.length; i++) {
        for (int j = 0; j < arr.length - 1 - i; j++) {
            if (arr[j] > arr[j + 1])
                ArraysTools.swap(arr, j, j + 1);
        }
    }
}

假设我们要排序的数组是已经是顺序的了，就没有再排序的必要。但是使用第一个方法进行排序的时候，即使数组已经是顺序的，但是还是会将所有的数进行比较交换。所以我们可以设置一个标志位来判断。是否已经排好序，这样可以减少无用的排序。

public void bubbleSort(int[] arr) {
    boolean flag = false;
        for (int i = 0; i < arr.length - 1; i++) {
            for (int j = 0; j < arr.length - 1 - i; j++) {
                if (arr[j] > arr[j + 1]) {
                    ArraysTools.swap(arr, j, j + 1);
                    flag = true;
                }
            }
            if (flag == false) {
                return;
            }
        }
}

3. 插入排序

实现思路

  • 将第二个数与前一个数开始进行比较，如果小（大），就进行交换；
  • 然后是第三个，将它与第二个数比较，如果小（大），就进行交换；
  • 在第三个与第二个数交换后，就再与第一个比较，如果小（大），就进行交换；
  • 类似上面的步骤，将后面所有的数都这样操作，直到所有的数都排好序

代码实现

public void insertSort(int[] arr) {
    for (int i = 1; i < arr.length; i++) {
        for (int j = i; j > 0; j--) {
            if (arr[j] < arr[j - 1])
            ArraysTools.swap(arr, j, j -1);
        }
    }
    long end = System.currentTimeMillis();
}

将上面的第二个for循环中if判断语句放在循环的判断条件中，可以进行一次对代码的优化。

4. 希尔排序

希尔排序是插入排序的一种优化版本。

实现思路

  • 它将插入排序分成一些段，然后对这些段进行排序。
  • 接着再将这些段的间隔都缩小，再进行排序，再然后将其间隔再缩小排序，直到所有数都排序完成。

代码实现

我们可以将数组以4为间隔来进行分段，在4分段排序后，将4除以2，使用2再排序，然后再除以2，就使用1来进行判断。当间隔为1时就和插入排序一样了。

public void shellSort(int[] arr) {
    for (int gap = 4; gap > 0; gap >>= 1) {
        for (int i = gap; i < arr.length; i++) {
            for (int j = i; j > gap - 1; j -= gap) {
                if (arr[j] < arr[j - gap])
                ArraysTools.swap(arr, j, j - gap);
            }
        }
    }
}

我们不可能对每个数组都是以4进行分段，而且遇到较大的数组的时候，使用4进行分段的时候，效率提高并不多，所以我么可以使用 Knuth 序列：

h = 1;
h = h * 3 + 1;

public void shellSort(int[] arr) {

    int h = 1;
    while (h <= arr.length / 3) {
        h = h * 3 + 1;
    }

    for (int gap = h; gap > 0; gap = (gap - 1) / 3) {
        for (int i = gap; i < arr.length; i++) {
            for (int j = i; j > gap - 1; j -= gap) {
                if (arr[j] < arr[j - gap])
                    ArraysTools.swap(arr, j, j - gap);
            }
        }
    }
}

5. 归并排序

实现思路

  • 将一个数组拆分成两部分；
  • 对数组左边进行排序；
  • 对数组右边进行排序；
  • 对左边的数组再拆分，并排序；右边同理
  • 排序过程中用到其他的额外空间

代码实现


public static void mergeSort(int[] arr, int left, int right) {
    if (left == right) return;

    int mid = left + (right - left) / 2;

    mergeSort(arr, left, mid);
    mergeSort(arr, mid + 1, right);

    merge(arr, left, mid + 1, right);
}

public static void merge(int[] arr, int leftPtr, int rightPrt, int rightBound) {
    int mid = rightPrt - 1;
    int[] temp = new int[rightBound - leftPtr + 1];

    int i = leftPtr;
    int j = rightPrt;
    int k = 0;

    while (i <= mid && j <= rightBound)
        temp[k++] = arr[i] <= arr[j] ? arr[i++] : arr[j++];

    while (i <= mid) temp[k++] = arr[i++];
    while (j <= rightBound) temp[k++] = arr[j++];

    for (int l = 0; l < temp.length; l++) arr[leftPtr + l] = temp[l];
       
}

6. 快速排序

在数组中找一个基数，遍历整个数组，将比基数小的数全部放到基数的左边，将比基数大的全部放到基数的右边。

实现思路

  • 将最右边的数定位基数，并设定左边和右边两个指针；
  • 将左指针指向数组的最左边，将右指针指向数组最后边第二个位置（即基数的前一个位置）
  • 左指针指的数和基数比较，若等于小于，则指针后移，将下一个数与基数比较，直到遇到大于基数的数停止；
  • 右指针的数和基数比较，若比基数大，则指针向前移动，将前一个数与基数比较，直到遇到小于基数的数为止；
  • 将左指针的数与右指针的数交换；
  • 一直进行比较交换，直到左指针和右指针相交停止；
  • 最后将基数与左指针指的数交换；

将以上步骤进行递归，则可以对整个数组进行排序。

代码实现


public static void quickSort(int[] arr, int left, int right) {
    if (left >= right) return;
    int mid = adjustArray(arr, left, right);
    quickSort(arr, left, mid - 1);
    quickSort(arr, mid + 1, right);
}

public static int adjustArray(int[] arr, int leftBound, int rightBound) {
    int x = arr[rightBound];
    int left = leftBound;
    int right = rightBound - 1;

    while (left <= right) {
        while (left <= right && arr[left] <= x) left++;
        while (left <= right && arr[right] > x) right--;

        if (left < right)
            ArraysTools.swap(arr, left, right);
    }

    ArraysTools.swap(arr, left, rightBound);

    return left;
}

7. 计数排序

计数排序很好理解，和字面的意思一样。通过计数来对整个数组进行排序。

实现思路

  • 设置一个统计数组，数组的大小为该原数组中最大的数减去最小数+1；
  • 再设立一个结果数组；
  • 对原数组进行遍历，将每个数减去最小数得到的数对应统计数组的下表，将该下表的数+1；
  • 将统计数组累加，得到每种数对应的最后一个位置；
  • 倒着遍历原数组，将统计数组的下表加上原来减去的最小数放在对应的位置，每次往前移动；

代码实现

public static int[] countSrot(int[] arr) {
    int max = findMaxMin(arr)[0];
    int min = findMaxMin(arr)[1];
    int[] result = new int[arr.length];
    int[] count = new int[max - min + 1];

    for (int i = 0; i < arr.length; i++) {
        count[arr[i] - min]++;
    }

    for (int i = 1; i < count.length; i++) {
        count[i] = count[i] + count[i - 1];
    }

    for (int i = arr.length - 1; i >= 0; i--) {
        result[--count[arr[i] - min]] = arr[i];
    }

    return result;
}

8. 基数排序

在基数排序中，使用了计数排序的内容。是对一个数字进行拆分，然后根据这些数的个位十位等进行一次排序。即把一个数的每一位当作一个基数。

实现思路

  • 和计数排序一样，设立统计数组和结果数组；
  • 找到整个数组中的最大值，算出最大值的位数；
  • 根据这个位数，从个位开始，进行计数排序；
  • 每位排序完成之后，将count数组清零，并且把得到的结果数组赋给原数组进行下一位的排序；

代码实现

public static void radixSort(int[] arr) {
    int[] result = new int[arr.length];
    int[] count = new int[10];

    int maxLength = (findMax(arr) + "").length();

    for (int i = 0; i < maxLength; i++) {
        int division = (int) Math.pow(10, i);
        for (int j = 0; j < arr.length; j++) {
            int num = arr[j] / division % 10;
            count[num]++;
        }

        for (int k = 1; k < count.length; k++) {
            count[k] = count[k] + count[k - 1];
        }

        for (int m = arr.length - 1; m >= 0; m--) {
            int num = arr[m] / division % 10;
            result[--count[num]] = arr[m];
        }

        System.arraycopy(result, 0, arr, 0, arr.length);
        Arrays.fill(count, 0);
    }
}
