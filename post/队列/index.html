队列 线性表 数据结构

在现实中，我们去银行办理业务的时候就需要排队。你来的早，排在前面，就会越早的离开。而在你排的队就是一个队列。 队列是个有序列表，它遵循先进先出的原则。

根据队列的先进先出的原则，我们就可以用代码来实现一个队列。

  1. 首先在一个ArrayQueue类中创建一个数组来代表一个队列
	private int[] arrayQueue; // 创建一个数组
    private int maxSize; // 队列的最大长度
    private int front; // 队头
    private int rear; // 队尾
  1. 然后我们要将这些信息进行初始化
	public ArrayQueue(int max) {
        this.maxSize = max; // 获取队列的长度用来初始化队列
        this.arrayQueue = new int[maxSize];
        this.front = -1; //一开始都是没有数据，所以头和尾都为-1
        this.rear = -1;
    }
  1. 然后是对队列的一些操作
	// 判断队列是否已满
    // 当队尾到达最大值时，队列已满（ 数组能储存的最大下标为 n - 1 ）
    public boolean isFull() {
        return rear == maxSize - 1;
    }

    // 判断队列是否为空
    // 当队头和队尾在同一位置时，则说明队列中间没有内容，即为空。
    public boolean isEmpty() {
        return front == rear;
    }

    // 添加数据
    // 添加数据时要先判断队列是否已满
    public void addQueue(int n) {
        if(isFull()){
            System.out.println("队列已满，不能加入数据");
            return;
        }
        rear ++;
        arrayQueue[rear] = n;
    }

    // 从队列中拿出数据
    // 同理要先判断队列里是否有数据
    // 根据先进先出原则来取数据
    public int getQueue() {
        if(isEmpty()) {
            throw new RuntimeException("队列为空，不能取出数据");
        }
        front ++;
        return arrayQueue[front];
    }

    // 遍历队列
    // 当队列为空时，直接显示队列为空队列
    public void showQueue() {
        if (isEmpty()) {
            System.out.println("队列为空，没有数据");
            return;
        }
        for (int i = 0; i < arrayQueue.length; i++)
            System.out.print(arrayQueue[i] + " ");
        System.out.println();
    }
