一个什么都不会的小菜鸟

Git简单使用

Aug 1, 2021
1 min read
Git 技巧

在本文中的所有命令操作都在git bash中进行

1、创建（克隆）仓库

1.1 初始化（创建）仓库：

git init

新建一个文件夹，该文件夹为你想创建的仓库。 在该文件夹中右键，打开git bash，输入该命令。不报错，则创建成功。

1.2 克隆仓库：

git clone <仓库链接>

进入一个文件夹，该文件夹应为所要放仓库的父文件夹。 打开git bash，输入该文件命令，等待克隆完毕即可。

2、提交推送仓库

2.1 添加文件到暂存区：

git add .
git add 文件名
git add 文件名 文件名 文件名 ……

当在本地仓库中做了修改之后，先要将本地仓库中的文件添加到暂存区。之后再来进行接下来提交推送操作。 这时需要使用 git add 命令来执行该操作。该命令使用有三种方式 第一种方式是将所有修改的文件添加到缓存区。 第二种方式是在 add 后面写需要添加的文件的名称 第三种方式和第二种方式相同，但是可以写多个文件名称来同时添加。

3、提交到本地仓库：

git commit -m "<这里是提交的注释>"

这一步的提交是将暂存区的修改提交到本地仓库。 在第三步完成之后来进行这一步操作。 输入指令回车后，不报错。则为提交成功。

4、推送到远程仓库：

git push

当在本地的操作都完成之后，便可以将本地仓库推送到远端的线上仓库。 使用push指令便可以将本地的仓库提交到远程的仓库。
